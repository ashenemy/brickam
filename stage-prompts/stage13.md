# ▶ STAGE 13 — AI-поиск
Выполни ТОЛЬКО Stage 13. Foundations §13.

**Сделай:** `feature/ai-search` + провайдеры в `ai-kit` (`LlmProvider`, `EmbeddingProvider`, за интерфейсами; ключи из env; mock в тестах).
- Эндпоинт `POST /ai/search`: текст задачи → LLM со **строгой JSON-схемой** `{projectType, themes:[{name, materialCategories[], keywords[]}]}` + ретраи при невалидном JSON → гибридный подбор (текстовый матч по keywords + **vector search** по `products.embedding` через Atlas Vector Search) → результат, сгруппированный по AI-темам, с пояснением темы.
- Генерация эмбеддингов товара при создании/правке (хук в catalog). Кеш частых запросов. Бюджет/троттлинг платных вызовов из конфига.

**Тесты:** юнит парсинга/валидации JSON-ответа (вкл. ретраи на моке); юнит сборки гибридного запроса; e2e «описание→темы→товары» на мок-провайдере.
