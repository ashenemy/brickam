# @brickam/idempotency

Идемпотентность мутирующих запросов Brickam. Хендлеры, помеченные
`@Idempotent()` (из `@brickam/server-kit`), при наличии заголовка
`Idempotency-Key` выполняются ровно один раз: повтор с тем же ключом и тем же
телом возвращает сохранённый ответ, повтор с другим телом — 409.

Ключи хранятся в коллекции `idempotency_keys` (уникальный индекс по `key`,
TTL-очистка 24ч). Глобальный `IdempotencyInterceptor` гейтит по метаданным +
заголовку и регистрируется как `APP_INTERCEPTOR`.
